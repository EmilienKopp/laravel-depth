# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0-beta - 2026-06-23

- Initial beta release.
- Added dependency tracing command: depth:trace suffix.
- Added JSON output option and file output option.
- Added route enrichment for controller entry points.
- Added root filtering with grep and uses options.
- Added static usage analysis for traits, extends/implements, typed properties, and method parameter types.
- Added and expanded unit test coverage for tracing and output formatting.
